# research

Research workflow skills — from literature review to automated experiment loops.

## Skills

| Skill | What it does |
|-------|-------------|
| `lit-review` | Structured paper review with analysis rubric |
| `log-research` | Experiment logging + published research logs |
| `write-research` | Guidelines for substantive research notes |
| `autoresearch` | Turn any question into an automated experiment loop on small models |

## Setup

Enable the plugin in Claude Code. `autoresearch` requires a coding agent with filesystem access and tmux for background loops.
