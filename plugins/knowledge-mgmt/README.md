# knowledge-mgmt

Strategic planning, vault tracking, and work logging for knowledge workers.

## Skills

| Skill | What it does |
|-------|-------------|
| `strategize` | Orient, plan day/week, stress-test plans — Socratic by default. Not for code planning. |
| `vault-tracking` | Commit vault/notes changes to git when substantial work accumulates |
| `log-work` | Timestamped entries in daily notes |

## Setup

Enable the plugin in Claude Code. Works best with a git-tracked markdown vault (Obsidian recommended but any markdown directory works).
