# Reference Projects

Other skills repos I recommend pointing your agents at. These complement what's in agent-utils — between them you get broad coverage of context engineering, agent workflows, and Claude Code patterns.

| Project | What it covers |
|---------|---------------|
| [Agent Skills for Context Engineering](https://github.com/muratcankoylan/Agent-Skills-for-Context-Engineering) | Context engineering workflows, planning, code review, documentation, architecture analysis |
| [Everything Claude Code](https://github.com/affaan-m/everything-claude-code) | Claude Code techniques, patterns, advanced configurations, prompt engineering |

## Adding a reference

```bash
git submodule add https://github.com/owner/repo.git reference/repo-name
```

Then add a row to the table above.
